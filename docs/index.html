Simple Text

Tag Name Description
<h1> Main Heading The top level heading of the webpage. You should only use this once on a page.
<h2> Sub Heading A subheading on the webpage. Used for topics under the main heading.
<h3> Sub Sub Heading Headings for topics under the <h2> headings.
<h4> ... <h6> Smaller Headings Titles for topics that are further nested below the above headings.
<p> Paragraph Used to display normal text such as the body of an article.

Emphasis

Tag Name Description
<b> Bold Changes the font weight of the text to be bold.
<strong> Strong Changes the font weight of the text to be bold but the browser also recognises the importance.
<i> Italic Changes the font style of the text to be italic.
<em> Emphasis Changes the font style of the text to be italic but the browser also recognises the emphasis.

Lists

Tag Name Description
<li> List Item An element that is displayed as one item in either an unordered or ordered list.
<ul> Unordered List A list that consists of <li> elements that are displayed in a bulleted list.
<ol> Ordered List A list that consists of <li> elements that are displayed in a numbered list starting from 1.

Font

Property Value Description
font-family Names of font families The value specifies the name of the font family for the text. These can be comma separated for fallback fonts if the previous ones are not available to the browser.
font-size Lengths, percentages, or keywords The value specifies the size of the font. The default size is 16px.
font-style normal, italic, oblique The value specifies the style of the font. The default style is normal.
font-weight normal, bold, bolder, lighter, 100 ... 900 The value specifies the weight of the font. The default weight is normal.
font-variant normal, small-caps The value specifies the variant of the font. The default variant is normal.
font Shorthand for font-style, font-variant, font-weight, font-size/line-height, and font-family The value specifies all the font properties in one declaration.
font-stretch normal, wider, narrower, ultra-condensed, extra-condensed, condensed, semi-condensed, semi-expanded, expanded, extra-expanded, ultra-expanded The value specifies the stretch of the font. The default stretch is normal.
font-size-adjust Number, none The value specifies the font size adjustment. The default adjustment is none.
